# ✅ To-Do Lista — Projekt w JavaScript

Prosta aplikacja To-Do List stworzona w czystym HTML, CSS i JavaScript. Pozwala na dodawanie, usuwanie i zarządzanie zadaniami. Projekt rozwijany jako część nauki front-endu.

---

## 🔧 Funkcjonalności

- ✅ Dodawanie zadań (opis + godzina)
- ✅ Usuwanie zadań
- ✅ Zmiana statusu wykonania zadania (np. zrobione/niezrobione)
- ✅ Obsługa interfejsu w oknie pop-up
- ❌ (W planach) Zapisywanie zadań w `localStorage`
- ❌ (W planach) Edycja zadań
